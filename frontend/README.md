## Getting Started
### Installation
1.  Clone the repository:
    ```sh 
    git clone <repo_url>
    ```
2.  Navigate to the project directory:
    ```sh
    cd careflow/frontend
    ```
3.  Install dependencies:
    ```sh
    bun i
    ```

### Running the App
Start the development server:
```sh
bun dev
```
The app will be available at `http://localhost:5173/`.
